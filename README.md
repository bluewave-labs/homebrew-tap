# Bluewave Labs Homebrew Tap

This is the official Homebrew tap for Bluewave Labs applications.

## Installation

```bash
brew tap bluewave-labs/tap
```

## Available formulas

### DodoShot

A beautiful, open-source screenshot tool for macOS.

```bash
brew install --cask bluewave-labs/tap/dodoshot
```

Or after tapping:

```bash
brew install --cask dodoshot
```

### DodoClip

A free, open-source clipboard manager for macOS.

```bash
brew install --cask bluewave-labs/tap/dodoclip
```

Or after tapping:

```bash
brew install --cask dodoclip
```
